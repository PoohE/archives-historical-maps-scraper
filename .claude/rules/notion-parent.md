# Notion: родительская страница для новых баз данных

## Правило

При создании **любой новой базы данных** в Notion через API —
всегда использовать страницу **HGIS** как родительскую.

```python
HGIS_PAGE_ID = "3830ba89-eabe-8136-9afd-fde0c7e20234"

# Создание новой базы:
api("POST", "https://api.notion.com/v1/databases", {
    "parent": {"type": "page_id", "page_id": HGIS_PAGE_ID},
    "title": [...],
    "properties": {...},
})
```

## Токены

- **Основной** (скрипты): `NOTION_TOKEN` из `Каталогизация/.env`
- **MCP (Pooh bot)**: в `settings.json` — для работы в чате

## Применяется к

- Всем скриптам в `Каталогизация/` и `Поиск онлайн архив/`
- При создании справочников, каталогов, аналитических таблиц
- Не применять для личных страниц пользователя вне проекта ИГИС
