# Правила работы с онлайн-источниками

## Если источник ничего не отдаёт — останавливай поиск по нему

При запуске поиска: если источник вернул 0 результатов на первых **3 комбинациях подряд** (территория × ключевое слово) — прекращать перебор оставшихся комбинаций для этого источника и переходить к следующему.

Логика в `run_search.py`:
```python
consecutive_empty = 0
MAX_EMPTY = 3  # после 3 пустых комбинаций подряд — пропускаем источник

if pos + doubtful == 0:
    consecutive_empty += 1
    if consecutive_empty >= MAX_EMPTY:
        print(f"[{source_id}] 3 пустых ответа подряд — пропускаем источник")
        health.issue(source_id, "0 результатов на первых 3 запросах — источник не отвечает на ключевые слова")
        break  # выход из цикла территорий для этого источника
else:
    consecutive_empty = 0
```

## Если источник ничего не нашёл — убирать из таблицы источников

После завершения поиска:
- Если по источнику **0 результатов за весь запуск** → пометить в Notion «Архивы»: `Статус доступа = Недоступен` + записать в «Проблема доступа» причину
- Убрать источник из `SOURCE_PRIORITY` в `run_search.py` до решения проблемы
- Восстанавливать только после ручной проверки и исправления searcher'а

## Известные проблемы источников (проверено 2026-07-03)

### RGO — geoportal.rgo.ru
- **Проблема:** URL `/search?q=` → 404. Текстового поиска нет.
- **Структура:** только каталог по разделам `/catalog/starinnye-atlasy-i-karty/`
- **Решение:** обходить каталог постранично, не использовать keyword-поиск
- **Статус:** searcher требует переработки; временно убрать из SOURCE_PRIORITY

### GPIB — elib.shpl.ru
- **Проблема:** неверный URL (`/ru/nodes?search=` вместо `/ru/nodes/search?query=`) и CSS-селекторы (`li.search-result` вместо `table tbody tr`)
- **Решение:** исправить в `_search_gpib()` — работает, 10 000+ результатов на «карта Калужская губерния»
- **Статус:** требует правки селекторов

### ЭтоМесто — etomesto.ru
- **Статус: ИСКЛЮЧЁН НАВСЕГДА**
- ЭтоМесто — геопортал-просмотрщик, не архив и не библиотека. Не является первичным источником исторических карт. Удалён из SOURCE_PRIORITY и из таблицы «Архивы» как нерелевантный.
