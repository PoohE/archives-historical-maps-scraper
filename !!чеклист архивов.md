**Цель:** найти исторические карты четырёх губерний — Калужской, Пермской, Смоленской, Ярославской.

**Легенда статусов:**
- ✅ готово/работает
- ⚠️ написано, не протестировано
- 🔄 в работе / нужны действия
- ❌ исключён / нет данных
- — нет скрипта

**Колонка «Проверка»** — финальная ручная верификация результатов (данные реально релевантны, попали в каталог).

**Колонка «Метод»** — рекомендация по стратегии:
- 🤖 = автоматический скрипт (надёжно, нет геоблока, простой API)
- 👤 = ручной поиск (геоблок / JS-рендер / сложная иерархия — риск > выгода)
- ❓ = неясно (не тестировалось, решим после первого запуска)

## Итого по приоритетам

### Протестировать без VPN (скрипты готовы)
```
python check_source.py neb               # НЭБ — баг исправлен, regex [^/]+
python check_source.py rsl               # РГБ — нужен тест без VPN
python check_source.py rosarchive        # Росархив — баг исправлен, verify=False + HTTPS
python modules/scrapers/searcher_garf.py --geo Калуж --debug    # ГАРФ Ф.1829
python modules/scrapers/searcher_gayao.py "карта Ярославская губерния" --year-to 1920 --debug
python modules/scrapers/searcher_gako.py "карта Калужская" --debug
python modules/scrapers/searcher_cgamos.py "карта Калужская" --debug
python modules/scrapers/searcher_gapk.py --list-funds --debug
```

### Нужен тест на российском IP (геоблок)
```
python check_source.py kaluga_lib        # Калужская КОНБ (переписать под WebIrbis64+)
python check_source.py smolensk_lib      # Смоленская СОУНБ
python check_source.py permkrai          # Пермская ПГКБ — нужен --check-node 33626
python modules/scrapers/searcher_sverdlovsk.py --check-ses   # ОПAC-Global
```

### Нужны действия вручную (DevTools)
- **РГИА** (#14): DevTools → Network → fgurgia.ru → найти AJAX URL
- **Госкаталог** (#17б): DevTools → Network → goskatalog.ru → найти JSON-запрос
- **ГАСО** (#20): Ф.259 изучить напрямую → `gaso-smolensk.ru/f-259-...`

### Ещё не написан скрипт
- Свердловская СОУНБ (#8а) — нужен тест с VPN/РФ-IP
- Калужская КОНБ (#6) — нужна переписка searcher_kaluga.py (P21DBN=EK, EK_FULLTEXT)

### Исключены навсегда
- РНБ история России (#4)
- Яндекс.Архивы (#17)
- ЭБИД (#15) и docs.historyrussia.org (#17в) — тот же шум
- СПбФ АРАН (#16) — требует логин
